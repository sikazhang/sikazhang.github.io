2018春招前端面试题总结

发表于 2018-04-21 | 分类于 个人生活 | 浏览量 次

春招至此告一段落，经历了漫长的3月和4月，海投了20多家公司，面了10好几家公司，拿到了腾讯和网易的实习offer。过程虽然充满辛酸和不易，但也带给我很多收获。闲话少叙，我将春招过程中一些常见面试题整理如下，希望对正在看文章的你有所帮助。

js

1.面向对象的三种状态，js中如何实现。
2.事件委托的好处
3.object属性说出一些
4.闭包：原理及应用场景
5.手写bind
6.promise怎么捕捉错误，如果捕捉到，还会执行下一个then么？
7.为什么script要放在body结束表现后面，如果我非要放在body结束标签之前呢？
8.es6新特性
9.localstorage怎么修改到期时间;sessionstorage
10.http浪费的是什么性能。
11.cookie
12.jsonp:手动实现，导致的安全问题
13.缓存
14.dns解析
15.http协议1.0/2.0，https原理
16.事件循环机制
17.实现即时通讯
18.哈夫曼编码
19.中间人安全攻击
20.prototype是什么、proto是什么，区别
21.1.js中引用2.js，如何将其合并为一个文件
22.csrf\xss\sql攻击
23.事件流
24.for each 和for..in和for..of的区别

css

1.css选择器
2.css兼容性
3.盒子模型
4.对行内元素设置padding和margin能否生效
5.css如何实现0.5边框
6.可以继承的css元素属性
7.css浮动
8.回流和重绘

框架与打包工具

1.vue-router的实现原理
2.require的实现原理
3.vue数据渲染原理
4.webpack是如何使模块运行起来的
5.如何用css设定div高度是宽度的一半,其中高度是自适应的。

以上就是一些我遇到很多次的性价比很高的面试题啦，好好掌握这些，基础部分应该就会给面试官一个比较不错的印象啦。

实习只是一个开始，最关键的还是要不忘初衷，认真踏实的学习前端的知识，好好打赢秋招这场战役。不为小有收获自满，更不能因为屡屡的挫败丧失了信心，加油！
